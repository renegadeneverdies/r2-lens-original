# Бот-трекер прогресса персонажей [R2 Original](https://r-2.online/rating/char)

Для работы нужно создать своего бота и канал для него с правами администратора. Скопировать файл `.env.example` и проставить значения переменным.
APP_ENV принимает значения development или production, в зависимости от того запускается ли программа с контейнера или на хосте. Значение production для контейнера.

Далее нужно заполнить документ levels в mongo. Это можно сделать запустив `ruby db/restore_levels.rb`.
Собирается через `docker compose up -d`. Инструкция в кронтаб запускает программу в 6 утра для чистоты эксперимента потому что квесты обновляются в это время.
